📋 项目概述

🎯 产品定位

JobGraph 是一款基于知识图谱的私密求职工具，核心卖点是隐私保护。

核心差异化: 在私有环境中安全求职，不留痕迹。相比 Boss 直聘等平台，JobGraph 不会追踪用户的浏览记录和求职意向。

HAS_REVIEW Company Review 公司收到的评价
HAS_PITFALL Company Pitfall 公司存在的坑点
MATCHES User Job 用户匹配的岗位

🌿 分支管理

📂 分支结构

master      # 管理员分支 (所有代码)
├── opensource  # 开源分支 (公开仓库)
└── user        # 用户功能分支 (开发)
分支 用途 内容 权限
master 管理员 所有代码 + 管理员工具 仅管理员
opensource 开源 用户代码 + 加密模块 公开
user 开发 用户功能开发 开发者

📝 总结

✅ 核心设计决策

  1. 隐私优先: 所有数据本地存储，不上传任何信息
  2. 知识图谱: 使用 Neo4j 存储实体关系，支持复杂查询
  3. License 保护: 分片加密 + 内存保护 + 设备绑定
  4. 低价策略: ¥9.9/月降低付费门槛
  5. 分支管理: master/opensource/user 三分支策略
