## 🎯 为什么选择 JobGraph？

### 🔒 隐私保护

**在 Boss 直聘等平台求职时，你是否担心过：**

| 风险 | 说明 |
|------|------|
| 👀 **浏览记录被追踪** | 你看了哪些公司，HR 都知道 |

| 特性 | 说明 |

| 🔐 **本地运行** | 所有操作在本地完成，不上传任何数据 |
| 👻 **无痕浏览** | 查看公司、搜索岗位不留任何痕迹 |
| 🚫 **无需注册** | 不需要手机号、邮箱等个人信息 |
| 💾 **数据自控** | 你的数据只属于你 |

## 🚀 快速开始

### 一键启动 (推荐)

```bash
# 克隆项目
git clone <repository-url>
cd jobgraph

# 一键启动 (自动安装依赖、启动数据库、导入初始数据)

# 访问 http://localhost:8504
```

### 其他启动方式

```bash

```

### 导入更多数据 (可选)

```bash
# 从 admin 仓库导入数据（推荐）
# 1. 先在 admin 仓库导出数据
cd ../jobgraph-admin
PYTHONPATH=. python3 scripts/export_data.py

# 2. 在 jobgraph 仓库导入数据
cd ../jobgraph
PYTHONPATH=. python3 scripts/import_from_admin.py --file ../jobgraph-admin/data/export/admin_data_YYYYMMDD.json

# 或从离线数据包导入
make sync-package FILE=data_package.zip

# 或从数据中心同步 (需要 Tailscale)
make sync-tailscale SERVER=http://数据中心地址:8000
```

**数据同步脚本说明**：

| 脚本 | 仓库 | 说明 |
|------|------|------|
| `scripts/export_data.py` | admin | 从 Neo4j 导出数据 |
| `scripts/import_from_admin.py` | jobgraph | 导入 admin 数据到 Neo4j |

## 📊 功能详情

### 简历上传与解析

- 支持 PDF、DOCX 格式简历
- 自动提取技能、工作年限、教育背景
- **100% 本地处理**：简历文件仅保存在你的电脑，不会上传到任何服务器
- **本地解析**：简历内容解析在本地完成，不会传出客户端
- **隐私过滤**：自动识别并过滤手机号、邮箱、身份证等敏感信息
- **匿名匹配**：匹配过程不使用姓名，只用技能、经验等非隐私信息

### 智能匹配

- 基于简历信息自动匹配岗位
- 多维度评分：技能匹配(40%) + 薪资匹配(25%) + 地点匹配(20%) + 经验匹配(15%)
- 匹配结果不足时，自动提示手动输入
- 支持手动输入职位信息进行匹配

### 简历优化（闭环优化）

当匹配度不高时，系统会自动分析差距并提供优化建议：

1. **差距分析**：识别技能、经验、学历等方面的匹配差距
2. **优化建议**：生成针对性的简历修改建议
3. **一键修改**：用户确认后自动修改简历
4. **重新匹配**：修改后自动重新匹配，直到满意

**隐私保护**：优化过程完全在本地进行，简历数据不会上传

### 岗位搜索

- 按关键词搜索
- 按地点筛选
- 按薪资范围筛选
- 显示公司风险等级

### 公司画像

- 公司基本信息
- 员工评价（优点/缺点）
- 薪资水平
- 风险评分（低/中/高/黑名单）

### 避坑指南

- 坑点类型：欠薪、PUA、996、内卷、裁员
- 严重程度评估
- 举报次数统计

### 薪资分析

- 职位薪资分布
- 行业薪资对比
- 城市薪资对比

## ❓ 常见问题

### Q: 数据从哪里来？

A: 应用内置了示例数据（34家公司、150+岗位、180+评价），开箱即用。你也可以：
- 自己导入数据
- 从数据中心同步最新数据（可选）

### Q: 必须连接数据中心吗？

A: **不需要！** 应用完全独立运行，内置数据足够使用。数据中心同步是可选的，用于获取最新数据。

### Q: 如何保证隐私？

A: 所有数据存储在你的本地数据库，不会上传到任何服务器。浏览记录、搜索历史都不会被记录。

### Q: 上传简历安全吗？

A: **非常安全！** 我们采用多重隐私保护：
1. **本地存储**：简历文件仅保存在你的电脑，不会上传到任何服务器
2. **本地解析**：简历内容解析在本地完成，不会传出客户端
3. **隐私过滤**：自动识别并过滤手机号、邮箱、身份证等敏感信息
4. **匿名匹配**：匹配过程不使用姓名，只用技能、经验等非隐私信息

> 💡 **即使简历包含隐私信息，也绝不会被传出去** — 所有处理都在本地完成

### Q: 匹配结果太少怎么办？

A: 可以使用「手动匹配」功能，手动输入期望职位、技能、地点等信息，系统会扩大搜索范围为您推荐更多岗位。

### Q: 支持哪些平台？

A: 支持 Windows、macOS、Linux。需要 Python 3.10+ 环境。

### Q: 如何获取最新数据？

A: 三种方式（可选）：
1. 使用内置示例数据（默认）
2. 从离线数据包导入
3. 从数据中心同步
